# 🚀 IntelliQ: Next-Gen GTM Intelligence Hub

**IntelliQ** is a high-fidelity, multi-agent platform designed for Product Managers, GTM Strategists, and Marketing Teams. It transforms scattered public-web data into concise, executive-grade product launch intelligence using a coordinated team of AI agents.

## ✨ Premium UI/UX Experience
IntelliQ isn't just a tool; it's a high-fidelity experience featuring:
- **Glassmorphism Design**: A sophisticated, layered UI with real-time blur effects and depth.
- **Premium Dark Mode**: A custom-crafted aesthetic using a deep slate palette and vibrant indigo accents.
- **Dynamic Animations**: Smooth "fade-in-up" transitions for content reveal and interactive feedback.
- **Responsive Layout**: Optimized for both high-end desktop monitors and mobile devices.

## 🤖 The Coordinated Intelligence Team
At the heart of IntelliQ is a three-agent team that coordinates in real-time to deliver multi-perspective insights:

| Agent | Expertise | Deliverables |
| :--- | :--- | :--- |
| **🔍 GTM Analyst** | Strategic Positioning | Competitive moats, pricing cues, channel mix, and strategic takeaways. |
| **💬 Sentiment Specialist** | Consumer Perception | Social media chatter, review themes, and positive/negative sentiment drivers. |
| **📈 Performance Lead** | Launch Metrics | User adoption, press coverage, market penetration, and qualitative buzz. |

## ⚡ Key Features
- **Real-Time Crawling**: Powered by Firecrawl for deep-web searches and asynchronous crawling.
- **Keyboard Shortcuts**: Navigate the intelligence hub like a pro:
  - `J` : Competitor Analysis
  - `K` : Market Sentiment
  - `L` : Launch Metrics
- **Executive Summaries**: Richly formatted Markdown reports featuring tables, call-outs, and actionable recommendations.
- **Full Traceability**: Every report includes a "Sources" section with direct links to the crawled data.

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/Shubhamsaboo/awesome-llm-apps.git
cd advanced_ai_agents/multi_agent_apps/product_launch_intelligence_agent
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Configure API Keys
You can provide keys via a `.env` file or directly in the secure sidebar UI:
```ini
OPENAI_API_KEY=sk-...
FIRECRAWL_API_KEY=fc-...
```

### 4. Launch the Platform
```bash
streamlit run product_launch_intelligence_agent.py
```

---

## 🕹️ Workflow
1. **Configure**: Enter your API keys in the sidebar (INTELLIQ will auto-detect them).
2. **Target**: Enter a company or product name in the main dashboard.
3. **Analyze**: Select an intelligence vertical (Competitor, Sentiment, or Metrics).
4. **Digest**: Review the executive-grade reports and strategic takeaways.
